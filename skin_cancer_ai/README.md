# Система ранней диагностики рака кожи на основе нейронных сетей

### Дипломный проект: Neural Network–Based System for Medical Image Analysis in Early Skin Cancer Detection

---

## О проекте

Данная система предназначена для автоматической классификации дерматоскопических изображений кожных новообразований с использованием сверточной нейронной сети MobileNetV2. Система обучена на датасете HAM10000 и способна распознавать 7 классов кожных заболеваний, включая меланому.

**Достигнутые результаты:**
- Точность (Accuracy): **80.8%**
- F1-Score (взвешенный): **80.6%**
- Recall меланомы: **51.5%**

## Установка и запуск

### Шаг 1 — Создание виртуального окружения

```bash
# Создать окружение на Python 3.11
py -3.11 -m venv venv

# Активировать (Windows)
venv\Scripts\activate

# Активировать (Mac / Linux)
source venv/bin/activate
```

### Шаг 2 — Установка зависимостей

```bash
pip install -r requirements.txt
```

### Шаг 3 — Загрузка датасета HAM10000

1. Зарегистрируйся на [kaggle.com](https://www.kaggle.com) и подтверди номер телефона
2. Установи Kaggle API: `pip install kaggle`
3. Скачай токен `kaggle.json` в `Settings → API → Create New Token`
4. Положи токен в папку `C:\Users\ИмяПользователя\.kaggle\`
5. Скачай датасет:

```bash
kaggle datasets download -d eliocordeiropereira/skin-cancer-the-ham10000-dataset -p data/
```

6. Распакуй архив в папку `data/`

### Шаг 4 — Визуализация данных

```bash
python visualize_data.py
```

Сохраняет в папку `results/`:
- `data_distribution.png` — распределение классов
- `sample_images.png` — примеры изображений
- `augmentation_demo.png` — демонстрация аугментации

### Шаг 5 — Обучение модели

```bash
python train_v4.py
```

Процесс обучения состоит из двух фаз:
- **Фаза 1** — обучение классификатора при замороженном backbone (25 эпох)
- **Фаза 2** — fine-tuning последних 30 слоёв backbone (20 эпох)

Время обучения на CPU: около 2–4 часов на полном датасете.

### Шаг 6 — Оценка модели

```bash
python evaluate.py
```

Выводит метрики по каждому классу и сохраняет:
- Confusion Matrix
- ROC-кривые
- Classification Report

### Шаг 7 — Предсказание для одного изображения

```bash
# Обычное предсказание
python predict.py data/HAM10000_images_part_1/ISIC_0025964.jpg

# Безопасное предсказание (порог меланомы снижен до 20%)
python predict_safe.py data/HAM10000_images_part_1/ISIC_0025964.jpg
```

### Шаг 8 — Запуск веб-приложения

```bash
python app/app.py
```

Открыть в браузере: **http://localhost:5000**

Возможности приложения:
- Загрузка изображения и получение диагноза
- Цветовая индикация уровня риска
- Вкладка тестирования — автоматическая проверка на случайных примерах из датасета

## Архитектура модели

Основу системы составляет архитектура **MobileNetV2**, предобученная на датасете ImageNet. Поверх неё добавлен классификатор:

```
MobileNetV2 (backbone, ImageNet weights)
    ↓
GlobalAveragePooling2D
    ↓
Dense(256, activation='relu')
    ↓
Dropout(0.35)
    ↓
Dense(7, activation='softmax')  ← 7 классов кожи
```

Обучение проводилось в два этапа: сначала замораживался backbone и обучался только классификатор, затем размораживались последние 30 слоёв для fine-tuning с уменьшенным learning rate.

---

## Ключевые технические решения

**Дисбаланс классов** — решён с помощью oversampling редких классов до среднего значения по датасету. Это позволило избежать коллапса модели в majority class (невус).

**Нормализация** — используется `preprocess_input` из MobileNetV2, приводящая значения пикселей к диапазону [-1, 1], что критично для корректной работы предобученных весов.

**Безопасный порог** — для меланомы введён пониженный порог срабатывания 20% (вместо стандартных 50%), чтобы минимизировать ложноотрицательные результаты.

---

## Ограничения системы

Система является вспомогательным инструментом скрининга и **не предназначена для замены врача**. Recall меланомы составляет 51.5%, что означает возможность пропуска части случаев. Финальный диагноз устанавливается только квалифицированным дерматологом.
